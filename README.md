🚀 MediaMind AI – Intelligent Content Personalization Platform

MediaMind AI is an AI-powered content generation platform that creates personalized and high-quality content using Large Language Models (LLMs). It helps users generate content like emails, LinkedIn posts, product reviews, and images based on structured inputs.

✨ Features
  📝 AI-based content generation (Emails, LinkedIn posts, Reviews)
  🎯 Personalized outputs based on user input (tone, audience, goal)
  🖼️ Image generation support
  🔗 REST API-based backend
  ⚡ Fast and optimized responses with caching
📂 Prompt and response storage using MongoDB
🛠️ Tech Stack
  Backend: Python, Django 
  Frontend: html,css
